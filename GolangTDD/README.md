## Golang TDD 중심으로 공부 시작  
## Simple.Tdd
- 대략 기본으로 우리가 할 수 있는 테스트 
## TDD
- 인테페이스를 이용해서 fake 모델을 만든다.
- 출력의 경우에도 출력을 테스트 할 필요 없이 출력하는 결과물만 비교하면 된다.
## Interface
- 인터페이스 학습 자료 
- 인터페이스 구현 코드 
- TDD를 위한 Mock 구현을 위한 인터페이스 공부 진행함
## Structure
- 구조체 학습 자료
- 구조체 구현 코드 
- 구조체를 활용하는 방법에 대해서 배우기 위해 공부 진행함
